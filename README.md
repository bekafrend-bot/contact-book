# Приложение «Контактная Книга»

Это полноценное веб-приложение для управления контактами, разработанное с использованием чистого HTML, CSS и JavaScript.

## Основные функции:
1. **Добавление контактов**: Форма для ввода имени, телефона, email и категории.
2. **Просмотр списка**: Динамическое отображение всех сохраненных контактов.
3. **Редактирование**: Возможность изменить данные существующего контакта.
4. **Удаление**: Функция удаления ненужных записей с подтверждением.
5. **Поиск**: Мгновенная фильтрация контактов по имени или номеру телефона.
6. **Категории**: Фильтрация контактов по группам (Семья, Работа, Друзья).
7. **Хранение данных**: Использование `localStorage` для сохранения данных в браузере (данные не пропадают после перезагрузки страницы).
8. **Адаптивный дизайн**: Приложение корректно отображается как на компьютерах, так и на мобильных устройствах.

## Структура проекта:
- `index.html` — разметка страницы.
- `style.css` — оформление и стили.
- `script.js` — логика работы приложения.

## Как запустить:
Просто откройте файл `index.html` в любом современном браузере.
